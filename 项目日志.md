# CPPReactor 项目日志

## 1. 文档用途
- 本文档仅用于记录当前工作进度与每日提交前后的对比更改。
- 本文档不包含代码改造建议，不替代设计文档。
- 当前按单人协作模式维护。

## 2. 版本号管理（仅日志内生效）
- 版本格式：`0.Y.Z`（在迁移完成前，主版本固定为 `0`）。
- 版本约束：所有版本必须 `< 1.0.0`。
- 升级规则：
  - `Z`（补丁）+1：仅补充记录、修正文案、无实际进度变化。
  - `Y`（小版本）+1：出现明确阶段进展或里程碑变更。
- 迁移未完成期间，不允许写入 `1.0.0` 或以上版本。

## 4. 当前基线快照
- 基线来源：`PROJECT_REFACTOR_MAP.md`
- 已形成主干：
  - `Channel` C++ 回调与 RAII 已建立。
  - `EventLoop` 任务队列、wakeup、分发框架已具备。
  - `EpollDispatcher` 基础后端可用。
  - `IOThreadPool` 创建、停止、join 主流程已具备。
  - `TcpConnection` C++ 类骨架已出现。
- 当前未收敛项：
  - `TcpServer` 接线与模式分支尚未统一。
  - 连接关闭语义尚未闭环。
  - `EventLoop` 的 `DELETE/MODIFY` 任务模型待收敛。
  - HTTP 解析链路尚未完成 C++ 适配。
